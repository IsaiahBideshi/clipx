# ClipX
A desktop application for capturing, trimming, and sharing game clips with friends. 
Built with Electron, React, and Supabase.

## Features
- <b>Video Trimming</b> — Cut clips down to its main purpose
- <b>YouTube Upload</b> — Clips are stored as unlisted YouTube videos via your connected Google account <br/>
  (I recommend creating a separate youtube channel for this) 
- <b>Friends System</b> — Add friends and share your clip library
- <b>Library</b> — Browse and view clips from people you follow
- <b>Share</b> - Copy the YouTube video link and share to your friends on any platform. 

## Tech Stack
| Layer    | Technology |
| -------- | ------- |
|Frontend | React + Vite |
|Desktop Shell | Electron |
| Auth / Database | Supabase (PostgreSQL)|
| Media Processing | FFMPEG |
| Storage | Youtube Data API v3 |
